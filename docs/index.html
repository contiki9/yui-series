概要

YUI-SERIESは各YUi-COMPONENTS達をextendで統合して、単体で使えるようにしたフレームワークです。

名前の由来はこちらです

Y = よくある
U = ユーザー
I = インターフェイス

YUi-COMPONENTSについて

YUI-COMPONENTSは単体で動くCSSではありません。 SCSSで作られていて、全てextendで作られています。

現在公開されているCOMPONENTSはサイドバーにあるもので確認できます。

使い方

git clone後 src/assets/scss/yui-series.scssを参考にあなたが使うYUi-COMPONENTSをextendで読み込ませるようにすれば設定完了です。

npmは落ち着いたら準備する予定です

カスタマイズ

ブランドネーム（接頭語）の変更

YUI-SERIESは予めブランドネームを変更できるようにしています。
デフォルト時には yuiをつけていますが $brandNameを変更すればあなたの好きなブランドネームを使えます。
